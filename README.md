# ARC LABS AFRIKA Website

A modern professional website for ARC LABS AFRIKA technology company built with React JS.

## 🚀 Features

- Modern React JS with functional components
- Tailwind CSS for responsive styling
- Framer Motion for smooth animations
- Lucide React icons
- Mobile-responsive design
- Professional startup/SaaS UI

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Create React App

## 🎨 Brand Colors

- Primary: #0B0F10 (dark black)
- Secondary: #F5B400 (yellow)
- Background: #FFFFFF (white)
- Accent: #2B2B2B (gray)

## 📦 Installation

```bash
npm install
```

## 🏃‍♂️ Running Locally

```bash
npm start
```

Opens the app at http://localhost:3000

## 🏗️ Building for Production

```bash
npm run build
```

Creates an optimized build in the `build` folder.

## 🚀 Deployment

### Vercel Deployment

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically detect and deploy the Create React App

The project includes:
- `vercel.json` configuration file
- Proper build scripts
- Optimized `.gitignore`

### Manual Deployment

1. Run `npm run build`
2. Deploy the `build` folder to your hosting provider
